# PERL 上課練習整理


## 語法

### 前綴變數
+ 註解：`#`
+ 變數：`$`
+ 陣列：`@`
+ 結尾：`;`
+ HASH：`%`

### if控制

| 數值 | 字串 | 意義             |
| ---- | ---- | ---------------- |
| ==   | eq   | euqal            |
| !=   | ne   | not euqal        |
| <    | lt   | less than        |
| >    | gr   | greater than     |
| <=   | le   | less or equal    |
| >=   | ge   | greater or euqal |
| <=>  | cmp  | <:-1, =:0, >:1   |
